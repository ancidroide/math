# Mathematics for Programming and Data Science

This repository contains a collection of mathematical summaries and notes created to support the study of mathematics alongside programming and data science.

The goal is to build a solid mathematical foundation, which is essential for understanding algorithms, machine learning models, and data analysis techniques.
